# JeuAventure

Un petit jeu textuel en Python — "Jeu dont vous êtes le héros" — organisé en paquet Python sous `src/jeuaventure`.

## Lancer le jeu

1. Assurez-vous d'avoir Python 3 installé sur votre ordinateur.
2. Ouvrez un terminal dans le dossier du projet.
3. Exécutez la commande suivante :

   python run_game.py

Le jeu se lancera dans le terminal.
